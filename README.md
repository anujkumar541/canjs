CanJS4 App
=====================

**A sample of product report with multiple filter.

## Requirements
* Git
* Node


## Common setup

To run this app follow the given steps

```bash
git clone https://github.com/virendrak4651/CanJS4.0.git
```

```bash
cd CanJS4.0
npm install
npm start
```

Browse the application on: http://localhost:4200/
